# Тема 3. Характеристики та порівняння оперативної пам'яті DDR4 та DDR5

## 1. Характеристики оперативної пам'яті типів DDR4 та DDR5, їх порівняння

Перехід від стандарту DDR4 до DDR5 є значним стрибком у розвитку технологій оперативної пам'яті. Основні зміни стосуються не лише швидкості, а й архітектури модуля, керування живленням та надійності.

### Основні відмінності:

| Характеристика | DDR4 (Double Data Rate 4) | DDR5 (Double Data Rate 5) |
| :--- | :--- | :--- |
| **Ефективна частота** | Зазвичай від 2133 МГц до 3200 МГц (JEDEC). XMP до 5000+ МГц. | Стартує від 4800 МГц (JEDEC). XMP вже досягає 8000+ МГц. |
| **Напруга живлення (VDD)** | 1.2 В (стандарт). Розгінні модулі часто використовують 1.35 В - 1.5 В. | 1.1 В (стандарт). Це сприяє енергоефективності, хоча швидкісні моделі можуть мати 1.25 В - 1.4 В. |
| **Архітектура каналів** | Один 64-бітний канал даних на модуль. | Два незалежних 32-бітних канали на один модуль (сумарно 64 біти). Це покращує ефективність доступу. |
| **Керування живленням** | Керування живленням здійснюється материнською платою. | Вбудований контролер живлення (PMIC) знаходиться безпосередньо на платі пам'яті. |
| **Максимальний об'єм чіпа** | До 16 Гбіт. | До 64 Гбіт (дозволяє створювати планки значно більшого об'єму, теоретично до 512 ГБ). |
| **ECC (корекція помилок)** | Тільки у серверній пам'яті. | On-die ECC (вбудована корекція помилок на кристалі) є частиною стандарту для підвищення надійності на високих швидкостях. |

### Порівняльна таблиця обраних моделей

| Параметр | Kingston FURY DDR4 | Kingston FURY DDR5 | Висновок |
| :--- | :--- | :--- | :--- |
| **Пропускна здатність** | 25.6 ГБ/с | 44.8 ГБ/с | DDR5 забезпечує **на 75% вищу** швидкість передачі даних. |
| **Таймінги (Latency)** | CL16 (низькі) | CL40 (високі) | DDR4 має менші затримки у тактах, але висока частота DDR5 частково компенсує це в реальному часі (наносекундах). |
| **Енергоспоживання** | 1.35 В | 1.25 В | DDR5 трохи енергоефективніша навіть при вищій частоті. |
| **Вартість** | Нижча | Вища (~ в 1.5-2 рази) | DDR4 залишається більш вигідним вибором для бюджетних систем, тоді як DDR5 — вибір на перспективу. |

### Висновок
Обраний комплект **DDR4-3200** є ідеальним рішенням для бюджетних та середніх збірок, де важливе співвідношення ціна/продуктивність.
Комплект **DDR5-5600** пропонує значно вищу пропускну здатність, що критично для нових процесорів, робочих завдань (архівація, рендеринг) та сучасних ігор, але вимагає відповідної материнської плати та більшого бюджет
